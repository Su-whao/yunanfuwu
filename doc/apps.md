
# APPS

## API

> 公共 API

## equipment

> 设备相关操作

- 查看设备信息
- 修改设备信息
  - manufacturer
  - model
  - groups
- 查看设备报警信息

## group

> 设备分组相关操作

- 查看分组
  - 查看所有分组（子账号）
  - 查看组内报警记录
- 修改信息
  - name
  - equipment（添加删除设备）

## record

> 记录相关操作

- 查看记录

## user

> 用户相关操作

- 用户登陆
- 用户登出
- 查看用户信息
- 查看操作记录
- 修改信息
  - name
  - username
  - password
  - child
  - address
  - describe
- 子账号操作
  - 查看所有子账号
  - 创建
  - 修改
  - 删除
  - 修改子账号组
- 父账号操作
  - 查看
